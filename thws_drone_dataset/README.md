### THWS Drone RLDS Dataset

This repository provides the RLDS-compatible dataset and tools used in our Drone Vision-Language-Action (VLAM) project.
The goal is to collect, structure, and preprocess drone navigation data from Unreal Engine and AirSim to support experiments on instruction following, gate-flying, and generalization in VLAM models.
